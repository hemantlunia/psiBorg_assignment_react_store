# React youStore App

A modern React application built with **React Router**, **React Query**, and **Tailwind CSS**, featuring authentication, protected routes, error boundaries, pagination...


---

##  Features

- Authentication (Login / Signup)
- Protected routes (Products, Product Details)
- Pagination
- Global error handling using Error Boundaries
- Toast notifications with `react-hot-toast`
- Centralized routing setup
- Clean provider architecture
- Tailwind CSS

---

## Tech Stack

- **React**
- **React Router**
- **@tanstack/react-query**
- **Tailwind CSS**
- **react-hot-toast**
